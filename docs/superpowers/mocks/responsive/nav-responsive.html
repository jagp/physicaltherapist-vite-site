Responsive nav — resize this window

What this demonstrates

One structural breakpoint at 768px (canonical). Desktop: logo + five links + gradient CTA, exactly today's look. Below 768: a 44px-target hamburger opens a full-width panel — built on <details>, so it's CSS-first, zero-JS, keyboard-operable out of the box, with the active page marked by the accent spine. Esc-close + click-outside are a 6-line progressive enhancement in the real build.

Fluidity everywhere else

Page gutters and the logo scale fluidly (clamp()), so nothing jumps between 768 and 1200 — the breakpoint handles structure only. Menu motion is wrapped in prefers-reduced-motion: no-preference.

Scroll — the header is sticky, panel overlays content.
